第 3 题 中等
关于过拟合，以下说法正确的是？
A
过拟合意味着模型在训练和测试集上都表现很好
B
过拟合可以通过增加正则化强度来缓解
C
过拟合只发生在深度学习模型中
D
增加训练数据量会加剧过拟合
答对了！+10 分
解析：过拟合是指模型在训练数据上表现优秀，但在未见数据上泛化能力差。正则化通过惩罚模型复杂度来缓解过拟合问题。常见的正则化方法包括 L1 正则化、L2 正则化和 Dropout 等。

第 5 题 困难
以下哪种方法不能有效防止过拟合？
A
增加训练数据量
B
使用 Dropout 正则化
C
增加模型复杂度
D
使用早停法（Early Stopping）
答错了
正确答案：C

解析：增加模型复杂度反而会加剧过拟合。模型越复杂，越容易 memorize 训练数据中的噪声。防止过拟合的有效方法包括：增加数据量、使用正则化（L1/L2/Dropout）、早停法、数据增强等。
