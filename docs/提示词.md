## 注意事项

UI 设计

```markdown
你是一个专业的 UI 开发者，擅长利用 HTML 开发 UI 模型，我会提供你一系列的资源，请你不留余力地完成任务。

## 提供的资源

@需求分析文档.md
@方案设计文档.md

## 需要完成的任务

请你帮我设计出一整套的 AI 学习闯关网站的 HTML	网页原型图

## 注意事项

1）你必须完全理解我给你的资源和任务，有任何不确定的地方必须及时找我人工确认，才能进行下一步
2）先给出一套 UI 原型草图和设计风格，等我人工确认，而不是直接启动开发
3）由于页面数量可能较多，建议你把每套相关的原型图（比如核心业务流程和拓展功能的区分）单独生成一个 HTML 文件
4）你必须先通过联网搜索，获取到市面上主流的同类网站的设计风格，给你提供参考
```

方案设计

````markdown
你是一位专业的独立开发者，擅长设计技术方案，我会给你提供一系列资源，请你帮我不遗余力地完成任务。

## 提供的资源

@需求分析文档

## 需要完成的任务

请你帮我完成详细的《方案设计文档》，包括技术选型、项目的开发流程和思路、以及其他方案设计文档的重点

## 注意事项

1）你必须完全理解我给你的资源和任务，有任何不确定的地方必须及时找我人工确认，才能进行下一步
2）由于小程序开发更新迭代非常快，你要重点帮我对比小程序的技术选型
3）由于功能非常多，请你分步骤实现，先实现最核心的功能，跑通核心业务流程（用户输入内容 => AI 生成题目 => 用户答题 => AI 生成分析报告），而不是先做用户登录注册这种需要数据库和其他依赖的内容

````

开发实现

````markdown
你是一个专业的 Java 全栈开发者，我会提供你一系列的资源，请你不留余力地完成任务。

## 提供的资源

@需求分析文档.md
@方案设计文档.md
@原型图

## 需要完成的任务

请你帮我完成 MVP 核心功能的完整前端 + 后端的开发

## 注意事项

1）你必须完全理解我给你的所有资源，有任何不确定的地方必须及时找我人工确认，才能进行下一步
2）先确认现阶段你需要完成的任务，并且给出你具体要执行的步骤，找我人工确认后，再进行开发
3）对于方案设计中提到的技术栈，你必须通过官网文档获取到最新的用法，确保编写的代码是正确的
4）UI 原型必须按照我提供的 UI 原型图来，不要自行发挥
5）你必须通过 TDD 自行测试所有的后端功能代码
````
